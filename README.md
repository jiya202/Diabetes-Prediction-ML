# Diabetes Prediction Using Machine Learning

## Objective

The objective of this project is to predict whether a patient has diabetes using machine learning techniques.

## Dataset

Pima Indians Diabetes Dataset

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-Learn
* Google Colab

## Model Used

Random Forest Classifier

## Accuracy Achieved

72.08%

## Project Workflow

1. Data Loading
2. Data Exploration
3. Data Visualization
4. Train-Test Split
5. Model Training
6. Prediction
7. Evaluation
8. Model Saving

## Conclusion

The Random Forest model was trained on the diabetes dataset and achieved an accuracy of 72.08%, demonstrating its capability to predict diabetes based on patient health information.
